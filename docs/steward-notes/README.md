# Steward Notes

These preserved all-caps markdown files are long-form audits, architecture memos, field notes, and session writeups that support ongoing stewardship of Astrid and Minime.

They were moved out of the repository root to keep the primary implementation docs easier to navigate:

- `CLAUDE.md` is the top-level operator/developer guide.
- `md-CLAUDE-chapters/` holds the current implementation chapters.
- `docs/steward-notes/` keeps the deeper historical and investigatory notes that remain valuable reference material.

When a note becomes implementation-critical, prefer reflecting the stable parts back into `CLAUDE.md`, the chapter set, or nearby source comments so the operational docs stay current.
